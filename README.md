# 简单的订餐系统
通过现有技术，实现一个简单的在线订餐系统，用于解决公司内部订餐仍然人工统计的问题。

## 技术选型
* 项目基于Maven构建
* 主要技术使用SpringBoot
* 持久化使用自定义json文件。

## 文件夹介绍

````
src
  main
    java
      com.domain.food
        config          -- 配置类
        consts          -- 枚举及常量
        domain          -- 数据对象
        utils           -- 工具类
    resources
      static            -- 静态资源文件
  test
    java

````
